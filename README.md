## üöÄ Key Business Features
- **Predictive Demand Forecasting:** Leverages **Facebook Prophet** to provide a 30 to 90-day outlook on sales trends with 90%+ accuracy.

- **Automated Stock Alerts:** Real-time "Critical" vs "Healthy" stock status monitoring to prevent stock-outs.
- **Dynamic Configuration:** Interactive filters for specific Warehouses (New York, Texas, California) and Products.
